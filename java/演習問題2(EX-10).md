## ポイント  
6行目の繰り返し文章である、  
int i = 1 ; i <= 9  
を実施したあとに、  
int j = 1 ; j <= 9  
へ移動し、そのあとに、
; j++  
に移動し、繰り返しを行う。    
int j = 1 ; j <= 9  ; j++  
で繰り返しできないまで進んだら、  
int i = 1 ; i <= 9  ; i++の ; i++ に戻り、  
また繰り返しが行われる。
